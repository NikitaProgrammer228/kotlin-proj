# Где находятся экспортируемые файлы

## Расположение файлов

Экспортируемые файлы (CSV и PDF) сохраняются в папку **`AccelerometerExports`** внутри директории Downloads.

### Для Android 10+ (API 29+)
**Путь:** `/storage/emulated/0/Android/data/com.accelerometer.app/files/Download/AccelerometerExports/`

**Особенности:**
- Файлы находятся во **внутренней директории приложения**
- Не видны в обычном файловом менеджере (без root-доступа)
- Доступны через:
  - Диалог в приложении (после экспорта показывается путь и кнопка "Открыть файл")
  - Android Studio → Device File Explorer
  - ADB команды

**Как найти через ADB:**
```bash
adb shell
cd /storage/emulated/0/Android/data/com.accelerometer.app/files/Download/AccelerometerExports/
ls -la
```

### Для Android 9 и ниже (API < 29)
**Путь:** `/storage/emulated/0/Download/AccelerometerExports/`

**Особенности:**
- Файлы находятся в **публичной папке Downloads**
- Видны в любом файловом менеджере
- Легко доступны для пользователя

## Как открыть файлы

### Вариант 1: Через приложение
После экспорта появится диалог с:
- Полным путём к файлу
- Кнопкой "Открыть файл" (откроет файл в подходящем приложении)

### Вариант 2: Через файловый менеджер
1. Откройте файловый менеджер на устройстве
2. Перейдите в папку **Downloads** → **AccelerometerExports**
3. Выберите нужный файл и откройте его

### Вариант 3: Через ADB (для разработчиков)
```bash
# Список файлов
adb shell ls -la /storage/emulated/0/Android/data/com.accelerometer.app/files/Download/AccelerometerExports/

# Копирование файла на компьютер
adb pull /storage/emulated/0/Android/data/com.accelerometer.app/files/Download/AccelerometerExports/measurement_2025-11-29_12-30-45.csv ./
```

### Вариант 4: Через Android Studio
1. Откройте **View** → **Tool Windows** → **Device File Explorer**
2. Перейдите в `/storage/emulated/0/Android/data/com.accelerometer.app/files/Download/AccelerometerExports/`
3. Правый клик на файле → **Save As...**

## Формат имён файлов

- **CSV:** `measurement_YYYY-MM-DD_HH-mm-ss.csv`
- **PDF:** `measurement_YYYY-MM-DD_HH-mm-ss.pdf`
- **PKT (правая нога):** `pkt_YYYY-MM-DD_HH-mm-ss_RIGHT.csv`
- **PKT (левая нога):** `pkt_YYYY-MM-DD_HH-mm-ss_LEFT.csv`

## Примеры путей

### Android 10+ (внутренняя директория)
```
/storage/emulated/0/Android/data/com.accelerometer.app/files/Download/AccelerometerExports/measurement_2025-11-29_14-30-00.csv
```

### Android 9 и ниже (публичная папка)
```
/storage/emulated/0/Download/AccelerometerExports/measurement_2025-11-29_14-30-00.csv
```

## Примечания

1. **Для Android 10+** файлы недоступны через обычный файловый менеджер из-за Scoped Storage. Используйте диалог в приложении или ADB.

2. **Если нужно сделать файлы доступными в публичной папке** для Android 10+, можно использовать MediaStore API (требует дополнительной реализации).

3. **Все экспортированные файлы** остаются на устройстве до удаления приложения или ручной очистки.
